# Волонтёрство — база исследований

Отдельный раздел сайта (`domain: volunteering`). Не путать с кастингами (оплачиваемая реклама) и конными лагерями.

## Аудитории

| Фильтр | Кто |
|--------|-----|
| 16+ / 17+ / 18+ | Подростки 16–18: ESC (с 18), обмен трудом, социальные проекты |
| 35+ | Взрослые 35+: Workaway, UN Volunteers, ESC Humanitarian Aid (до 35) |

## Подкатегории на сайте

- **ESC** — European Solidarity Corps (18–30, гуманитарная линия до 35)
- **Волонтёрский обмен** — HelpStay, Workaway (обмен трудом на проживание)
- **Социальные проекты** — gap year, community theatre, Masa Gap
- **35+** — meta-категория для взрослых программ (`audience: adult`)
